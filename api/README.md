# Optimizer api

An api to facilitate the optimization app

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements.

```bash
pip install -r requirements.txt
```

## Usage

```python
cd api

uvicorn main:app --reload
```
